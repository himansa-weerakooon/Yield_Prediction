# Crop Yield Prediction

This project focuses on analyzing remote sensing and meteorological data to understand how crop yields fluctuate based on environmental conditions.

The process involved comprehensive Exploratory Data Analysis (EDA) and Advanced Data Analysis (ADA) to handle the dataset.
The process involved comprehensive data cleaning and preprocessing to handle inconsistencies, including eliminating variables that were linear combinations of each other.
Various regression models were built and the best was chosen to ensure accurate results.
We specifically examined the impact of variables such as NDVI derived from satellite images, soil moisture, rainfall, and temperature to see how they affect agricultural output.
Finally, we developed a data product that offers crop yield predictions based on given input data.

Source: [Kaggle - Crop Yield Prediction](https://www.kaggle.com/datasets/nishchalchandel/crop-yield-prediction)
